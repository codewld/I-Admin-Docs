# iDataContainer - 数据容器组件

# 特点

  • 自动查询数据并向外提供
  • 简化数据查询，省去手动编写查询发起、Loading 设置、结果赋值、错误处理的麻烦

# Props

codenamenote

immediate是否立即请求默认为 true
cacheConf缓存配置传入包含 key 属性的对象，key 应该唯一
